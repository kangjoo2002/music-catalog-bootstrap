# Roadmap

- Better install paths
- Release ZIP validation in CI on both Windows and Linux
- More polished example outputs for apply mode
- More failure-mode coverage around direct apply

- Additional output targets beyond MySQL and PostgreSQL

## Not Planned Right Now

- Broad multi-entity ingestion in one jump
